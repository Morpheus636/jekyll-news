About Jekyll-News

Jekyll-News is an all-in-one, responsive newspaper theme for Jekyll. Jekyll-News is 100% static, built entirely with HTML and CSS. The only included JavaScript is for optional features like comments, analytics, and ads.

Features

  • Fully static
  • Classic, responsive design
  • Home page template
  • About page template
  • Staff Profiles
  • Article template complete with a featured image and cutline
  • Built-in support for Disqus comments
  • Built-in support for Google Analytics
  • Built-in support for Google AdSense
  • Optional, easy to modify sidebar
